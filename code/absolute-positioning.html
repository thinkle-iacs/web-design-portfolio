Absolute Positioning

Modeling

  • You use a position: absolute, or position: fixed to position elements in an overlapping way on the page.
  • You use position: relative or the equivalent on an ancestor of your absolute positioned item in order to make your absolute positioning code as simple as possible (i.e. you choose a logical reference point for absolute positioning)
  • You layer items correctly to create effective overlaps, either by using the order of elements or by using z-index
