This repository contains descriptors for skills that my introductory web design students need to demonstrate.

This itself is not meant to be beautiful :)

Skills are divided into categories based on the strand system our science department uses:

  • mod - Modeling. Understanding the abstractions behind web pages.
  • eu - Enduring Understanding. Used for long term comprehension of concepts. In Web Design, I use this strand for design skills.
  • ps - Problem Solving. In web design I use Problem Solving for finding a way through challenging design or coding problems.
